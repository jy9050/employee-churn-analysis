# Employee Attrition Analysis Dashboard – Green Destinations
This project analyzes employee attrition trends using Tableau.

## Key Insights
- Overall Attrition Rate: 16.12%
- Higher attrition in Sales & R&D departments
- Most attrition in age group 28–36
- Males have higher attrition than females
