# 📊 Employee Churn Analysis

This project analyzes employee attrition and builds a predictive model to identify key factors driving employee churn.

It combines data analysis, feature engineering, and machine learning to generate actionable business insights.

## 📂 Dataset
Employee dataset containing demographic, job-related, and satisfaction features.

## 📊 Analysis
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Feature engineering

## 🤖 Machine Learning
- Model: Logistic Regression
- Target: Attrition (Yes/No)

## 💡 Key Insights
- Employees working overtime are more likely to leave
- Low job satisfaction strongly impacts attrition
- Salary plays a key role in retention

## 📈 Conclusion
Organizations can reduce attrition by improving employee satisfaction and managing workload effectively.
